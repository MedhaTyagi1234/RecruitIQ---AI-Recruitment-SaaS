# RecruitIQ 🚀  
### AI Recruitment SaaS 
RecruitIQ is a modular AI-based recruitment SaaS built using Python and Pandas.  

## 📦 Module 1 Overview

Module 1 handles the **data ingestion and preprocessing layer**, which is the foundation for all future intelligence modules.

## ⚙️ Features

### 📁 File Handling
- Upload CSV, XLS, XLSX files
- Read full dataset using Pandas

### 📊 Dataset Processing
- Detect total rows and columns
- Extract column names
- Column indexing for selection
- Create analysis-ready dataset

### 🛡️ Data Validation
- File size validation (≤ 20MB)
- Empty file detection
- Unsupported file format handling
- Corrupted file handling

## 🧠 Why this module matters

Recruitment AI systems depend heavily on clean and structured data.  
This module ensures that all incoming candidate datasets are validated and standardized before analysis.

## 🛠️ Tech Stack

- Python
- Pandas
- Google Colab (Development Environment)
